程序功能：

1. releasecheck
本程序用来检验条目是否已发布。原理是利用了发布平台的检索系统的高级检索功能检索词条，只进行标题精确检索。鉴于检索系统会将检索结果排序，精确匹配的放在前面，所以我们只保存了第一页的前8项结果，就可以显示条目是否已发布。
使用方法：
a.要检索的条目列表，填入template.xlsx文档文件中。请不要改变列的位置和名称。
b.双击releasecheck.bat文件，运行程序。
c.检索结果保存在data目录下。

发布日期：
2021.07.27

2. entrydownloader
因为发布平台限制拷贝，本程序用来辅助下载条目文本内容。
使用方法：
a.在平台检索到条目，拷贝条目页面的URL填入URLs.xlsx文档中。请不要改变列的位置和名称。
b.双击entrydownloader.bat文件，运行程序。
